# Aluluei-packs

Aluluei — data packs
  
  Public region data packs for the the navigation app. Downloaded by the app you choose **Full offline** mode

  Each region is published as a **GitHub Release** (see the *Releases* tab), not as
  files in this repo. A release contains:

  | File | What it is |
  |------|------------|
  | `manifest.json` | Pack index — file list, sizes, SHA-256, coverage bbox |
  | `gh_nl.zip` | GraphHopper routing graph (offline turn-by-turn) |
  | `netherlands-roads.pmtiles` | Drivable-roads vector tiles |
  | `nl_pois.db` | Points-of-interest database |
  | `nl_addresses.db` | Address / street-search database |

  ## Data source
  Built from [OpenStreetMap](https://www.openstreetmap.org/) data.
